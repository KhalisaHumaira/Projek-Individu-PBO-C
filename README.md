**Program Manajemen Penumpang Bus**

Program ini dibuat menggunakan Java untuk mensimulasikan sistem penumpang pada bus Trans Koetaradja.

**Penumpang akan ditempatkan berdasarkan aturan:**
1. Prioritas ( lansia (> 60 tahun), anak-anak (< 10 tahun), ibu hamil )
2. Kursi biasa
3. Berdiri jika kursi penuh

Saldo awal penumpang sebesar 10.0000 dan penumpang otomatis dipotong Rp 2000 saat berhasil naik, dan pendapatan bus akan bertambah.

**Kapasitas:**
1. Kursi prioritas (4)
2. Kursi biasa (16)
3. Berdiri (20)
   
**Total: 40 penumpang** 

**Cara Menjalankan:**
Compile semua file lalu jalankan.
